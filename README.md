# Snake-Xenzia
# 🐍 Snake Game (Java Swing)

A classic **Snake Game** built using **Java Swing** with a graphical interface, difficulty levels, and a high score system.
The project demonstrates GUI development, event handling, and basic game logic in Java.

---

## 📌 Features

* 🎮 Classic Snake gameplay
* ⚡ Difficulty selection (Easy / Medium / Hard)
* 🏆 Global high score tracking
* 👤 Username input before starting the game
* 🖥 GUI built with Java Swing
* 🔄 Smooth snake movement and collision detection
* 🍎 Random food spawning

---

## 🛠 Technologies Used

* Java
* Java Swing (GUI)
* AWT Event Handling
* Object-Oriented Programming (OOP)

## ▶ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/snake-game.git
```

### 2️⃣ Navigate to the project folder

```
cd snake-game
```

### 3️⃣ Compile the program

```
javac *.java
```

### 4️⃣ Run the game

```
java SnakeApp
```

## 🎮 Controls

| Key           | Action                         |
| ------------- | ------------------------------ |
| ⬆ Up Arrow    | Move Up                        |

| ➡ Right Arrow | Move Right                     |
| Enter         | Return to Menu after Game Over |

## ⚙ Difficulty Levels

| Level  | Speed    |
| ------ | -------- |
| Easy   | Slow     |
| Medium | Moderate |
| Hard   | Fast     |

Difficulty is selected before starting the game.

---

## 🧠 Game Rules

* The snake grows after eating food.
* Each food increases the score.
* The game ends if the snake:

  * Hits the wall
  * Collides with itself
